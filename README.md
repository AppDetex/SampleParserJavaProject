# SampleParserJavaProject

A utility to scrape Google Play app pages and return parsed data as json.


## Tested using:
- OpenJdk 11.0.1
- Maven 3.6.0


## Usage

Build it:

- ```mvn clean package```

Run it:
- ```java -jar ./target/sampleparserjavaproject.jar "url-of-google-play-app-to-scrape"```
